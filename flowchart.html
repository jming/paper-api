Process

Click on an existing box to add a new step to the procedure that you want to communicate. The three main types of steps you can include are:

  • Question. Basic question to determine the direction of the next step. For example, "How long has the child had a fever?"
  • Answer. This is an answer option to each of the questions. For example, "Less than 7 days" or "7 days or more" would be answers to the first question.
  • Instruction. This is an instruction of an action to take once the state has been determined by the answers give. This is terminal and the flowchart procedure should end here. For example, if a child has had a fever for "7 days or more," it is advised that "The child should be brought to the nearest community health center."
See the resources page to see some examples of flowcharts you can construct.
